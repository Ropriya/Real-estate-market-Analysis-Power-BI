# Real Estate Report Dashboard

## Overview
This Power BI dashboard provides comprehensive analytics on real estate properties, including sales trends, revenue analysis, property distribution by country, and key performance metrics.

## Dashboard Components

### Properties by Name of Country
- Interactive world map showing property distribution across 11+ countries
- Top markets: China (843K), United States (638K), India (612K)

### Team Members
- Profile cards displaying team information:
  - Aria Taylor (1618857)
  - Emmy Johnson (9705688)
  - Isabelle Anderson (786195)
- Visual representation with member photos and ID numbers
- Calendar view showing 2025, 2026, 2027

### Property Showcase
- Featured property image: Bridge Resort building
- High-quality visual representation of real estate assets

### Sum of Expense and Quarter
- Area chart showing quarterly expense trends
- Timeline progression from Q1 to Q4
- Expense range from 0.0M to 4.0M

### Revenue by Month Name
- Bar chart displaying monthly revenue distribution
- Clear visualization of revenue patterns throughout the year

### Key Performance Indicators
Four major KPI cards:
1. **Expense**: 20M
2. **Revenue**: 26M
3. **Properties**: 6M
4. **Sold Properties**: 26

### Name of Country and Revenue
- Horizontal bar chart showing country-wise revenue distribution
- Detailed breakdown of revenue by geographic location

## Data Sources
- **Calendar**: Date and time dimension data
- **Customer details**: Client information and demographics
- **Measure Table**: Calculated metrics and KPIs
- **Property details**: Property specifications and attributes

## Features
- **Interactive Filters**: Available on both page-level and report-level
- **Cross-report Drill-through**: Enabled for detailed analysis
- **Keep all filters**: Maintains filter state across interactions
- **Geographic Visualization**: Interactive map for property distribution
- **Team Collaboration**: Team member profiles and contact information

## Key Insights
1. **Global Reach**: Properties distributed across 11+ countries
2. **Strong Revenue**: 26M in total revenue with 20M in expenses
3. **Large Portfolio**: 6M properties with 26 sold properties tracked
4. **Top Markets**: China, United States, and India lead in property counts
5. **Quarterly Trends**: Clear expense patterns across quarters
6. **Profitability**: Revenue exceeds expenses, indicating positive margins

## Usage Instructions
1. Click on the map to filter data by specific countries
2. Use the quarter slider to analyze expense trends over time
3. Navigate through different pages using the page tabs at the bottom
4. Apply filters using the right-side panel for customized analysis
5. Hover over visualizations for detailed tooltips and insights
6. Click on team members for additional contact information

## Data Refresh
Ensure data sources are properly connected and refresh the report to see the latest real estate data and analytics.
